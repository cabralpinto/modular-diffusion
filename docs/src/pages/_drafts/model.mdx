# diffusion.Model

## load()

### Parameters

- `path: str`: Path to model.

## save()

### Parameters

- `path: str`: Path to model.

## train()

### Parameters

- `epochs: int = 1`: Number of epochs.

- `Iterator[float]`: Training losses.
